# Wimp Membership — Napkin Math

Interactive calculator for a Wimp Decaf membership: members pay a yearly fee,
then buy bags at **cost + margin** instead of retail. Drag the dials.

**Live preview:** https://wimpdecaf.github.io/membership/

Single self-contained `index.html` — no build, no dependencies. Open it in any
browser or edit it directly. Both org owners (whale, liamtucker) are admins.
